diocp5
======

设置环境变量
DIOCP5_HOME=E:\workspace\diocp-v5

搜索路径
$(DIOCP5_HOME)\Source


目录说明:
  samples                        下面是各种DEMO
  source                         目录下面是源代码
  
可以先从下面的DEMO中了解diocp的工作原理
  samples\ECHO
  samples\simple
  samples\StringDEMO
  
从那里得到:
 git更新(推荐同步更新)
  1.https://github.com/ymofen/diocp-v5
  2.https://git.oschina.net/ymofen/diocp-v5
 最新版zip下载
  https://codeload.github.com/ymofen/diocp-v5/zip/master
